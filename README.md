# Playwright Python Frameworks
### Features:
* Test web search UI
* Test rest api endpoints
* Data driven tests
* Run test in parallel
* Generate HTML report
* Basic rest api server with Flask


## installation
```shell
pip install -r requirements.txt
playwright install
```

## run tests
```shell
cd scripts
./run_tests.sh
# or
pytest --headed 
```

## start rest server
```shell
python main.py
```
